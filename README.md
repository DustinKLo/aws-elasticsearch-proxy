## Loading configurations from settings.yml

```
# settings.yml
log_level: warn # debug, info, warning, error
log_file_path: /path/to/log/file.log

host: http://localhost:9200
http_scheme: http
verify_ssl: false

service: es
aws_region: us-west-2
```

## Run app
```
go run main.go
```
